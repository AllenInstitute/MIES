AnalysisBrowser¶

The AnalysisBrowser allows to load acquired data from Igor Experiment files (pxp and uxp) and NeuroDataWithoutBorders files (nwb, v1 and v2) into one Igor Experiment. The user can then open sweeps from multiple experiments in one SweepBrowser, see DataBrowser, inspect the labnotebook entries of these sweeps and view the TPStorage data.
