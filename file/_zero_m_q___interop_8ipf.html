File ZeroMQ_Interop.ipf¶

Flags for zeromq_set()

** Define ZeroMQ Constants (for back-compatibility) for Igor8 case. **

const double ZMQ_SET_FLAGS_DEFAULT = 0x1¶

Sets the default flags (no debug, no ipv6, busy wait on receive)

const double ZMQ_SET_FLAGS_DEBUG = 0x2¶

Enable debug output.

const double ZMQ_SET_FLAGS_IPV6 = 0x4¶

Enable ipv6 support.

const double ZMQ_SET_FLAGS_NOBUSYWAITRECV = 0x8¶

Don’t do busy waiting on zeromq_server_recv() and zeromq_client_recv() instead immediately return if no messages are available.

const double ZMQ_SET_FLAGS_LOGGING = 0x10¶

Log incoming and outgoing messages.
